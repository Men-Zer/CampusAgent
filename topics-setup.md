# 为仓库添加话题标签

为了让更多人发现 CampusAgent，请在 GitHub 仓库设置中添加以下 Topics：

## 必需标签（至少选 3-4 个）

| 标签 | 说明 |
|------|------|
| `ai-agent` | AI 智能体，当前热门话题 |
| `rag` | 检索增强生成，RAG 的热度持续上升 |
| `langgraph` | LangChain 的 Agent 框架，开发者常关注 |
| `ollama` | 本地 LLM 运行工具，社区活跃 |
| `campus` | 校园场景，中文用户搜索关键词 |

## 推荐标签（选加）

| 标签 | 说明 |
|------|------|
| `chromadb` | 向量数据库 |
| `fastapi` | Python Web 框架 |
| `llm` | 大语言模型 |
| `educational` | 教育类项目 |
| `chatbot` | 聊天机器人 |
| `qwen` | 通义千问模型 |
| `python` | Python 项目 |

## 设置方法

1. 打开仓库页面：https://github.com/Men-Zer/CampusAgent
2. 点击右侧 ⚙️ **About** 区域右上角的齿轮图标
3. 在 **Topics** 输入框中逐个输入标签，按回车添加
4. 点击 **Save**
