# Setting GitHub Topics for AgenticRAG

Topics help people discover your repository by category. Here's how to set them:

1. Go to https://github.com/hyx1249207016-netizen/AgenticRAG
2. Click the gear icon next to "About" on the top right
3. Under "Topics", add these tags (space-separated):

```
rag langgraph ollama langchain chromadb fastapi agent chatbot llm ai-agent
```

4. Click "Save changes"

These tags ensure your project appears when users search for relevant keywords.
