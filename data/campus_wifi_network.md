# 校园网络与设备使用指南

## 校园 WiFi 连接

大多数高校的校园 WiFi 连接方式：

### 连接步骤

1. 打开设备 WiFi，搜索校园无线网络（常见名称：学校缩写+WiFi、Campus-Net、或运营商名称）
2. 连接后浏览器通常会自动弹出认证页面（Portal 认证）
3. 输入学号和校园网密码（初始密码一般为身份证后 6 位或学号，登录后务必修改）
4. 认证成功后即可上网

### 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 连上 WiFi 但打不开网页 | DNS 解析失败或认证过期 | ① 断开 WiFi 重连 ② 浏览器访问任意网址看是否弹出认证页 ③ 打开命令行执行 `ipconfig /flushdns` |
| 认证页面弹不出来 | 浏览器缓存或 IP 地址未获取 | ① 手动在浏览器输入 1.1.1.1 或学校官网试试 ② 关闭浏览器的"安全 DNS"功能 ③ 忘记网络后重新连接 |
| 显示"账号已在其他地方登录" | 多设备同时登录 | ① 在认证页面点"下线" ② 修改校园网密码后重试 |
| WiFi 信号满格但网速慢 | 接入人数过多 | ① 换到人少的教学楼或图书馆 ② 优先使用有线网口（宿舍一般有） |
| 连接提示"无法获取 IP 地址" | DHCP 地址池满 | 等待 2-3 分钟后重试，或手动设置 IP 地址 |

## 电脑磁盘空间管理

大学生容易遇到 C 盘空间不足的问题：

- **查看磁盘使用**：右键 C 盘 → 属性 → 磁盘清理，系统会自动列出可清理的临时文件
- **迁移微信/QQ 文件**：微信和 QQ 的聊天记录和文件默认存在 C 盘，在设置中修改保存位置到 D 盘
- **清理下载文件夹**：Windows 默认下载路径在 C:\Users\用户名\Downloads，定期清理大文件
- **使用磁盘分析工具**：免费工具如 SpaceSniffer 可以快速定位大文件

## 免费软件获取

大学生可以免费获取的正版软件：

- **Office 365**：用学校邮箱（@xxx.edu.cn）在 office.com 注册教育版，免费使用 Word、Excel、PowerPoint
- **WPS 校园版**：部分学校提供免费 WPS 会员
- **JetBrains 全家桶**：用学校邮箱在 jetbrains.com/student 申请，免费使用 PyCharm、IntelliJ IDEA 等
- **GitHub Student Pack**：在 education.github.com 用学校邮箱申请，含大量免费开发工具和学习资源
