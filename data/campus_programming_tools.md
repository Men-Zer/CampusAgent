# 编程入门与开发工具

## 入门语言选择
| 语言 | 适合方向 | 学习难度 |
|------|----------|---------|
| Python | 数据分析/AI/自动化 | 低 |
| JavaScript | 前端/Web开发 | 低 |
| C/C++ | 竞赛/底层/嵌入式 | 中 |
| Java | 企业级/Android | 中 |

## 开发环境
| 工具 | 用途 | 获取 |
|------|------|------|
| VS Code | 通用代码编辑器 | 免费 |
| PyCharm | Python IDE | 教育版免费 |
| IntelliJ IDEA | Java IDE | 教育版免费 |
| Git + GitHub | 版本控制 | GitHub教育包免费 |

## 学习路线
1. 选择一门语言，看完基础语法
2. 做一个小项目（如：计算器、天气查询、个人博客）
3. GitHub 上找开源项目模仿
4. LeetCode 刷题（找实习面试必备）
