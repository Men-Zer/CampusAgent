# 大学压力管理

## 压力来源
- 学业压力（考试、作业截止日期）
- 人际压力（室友矛盾、社交焦虑）
- 未来压力（考研、就业的不确定性）
- 经济压力（学费、生活费）

## 应对方法
| 方法 | 说明 |
|------|------|
| 分解任务 | 把大作业拆成每天做一点 |
| 番茄钟 | 只要开始做25分钟，就不焦虑了 |
| 找人聊聊 | 和朋友/辅导员/家人倾诉 |
| 运动 | 跑完三公里，压力少一半 |
| 做能控制的事 | 你不能控制考研分数线，但能控制今天背多少单词 |
