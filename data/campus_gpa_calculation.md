# 大学绩点（GPA）计算与提升指南

## 绩点计算方式

大多数中国高校采用 4.0 或 5.0 绩点体系：

| 百分制 | 等级 | 绩点（4.0） | 绩点（5.0） |
|--------|------|------------|------------|
| 90-100 | A/优秀 | 4.0 | 4.0-5.0 |
| 85-89 | A- | 3.7 | 3.7 |
| 82-84 | B+ | 3.3 | 3.3 |
| 78-81 | B | 3.0 | 3.0 |
| 75-77 | B- | 2.7 | 2.7 |
| 72-74 | C+ | 2.3 | 2.3 |
| 68-71 | C | 2.0 | 2.0 |
| 64-67 | C- | 1.5 | 1.5 |
| 60-63 | D | 1.0 | 1.0 |
| <60 | F/不及格 | 0 | 0 |

## GPA 排名计算

GPA = Σ(课程绩点 × 课程学分) / Σ课程学分

如：
- 高数（4学分，90分 → 4.0）= 16.0
- 英语（2学分，85分 → 3.7）= 7.4
- GPA = (16.0 + 7.4) / (4+2) = 3.9

## 提高绩点的方法

- 平时分占 30%-50%，认真上课回答问题、交作业、不缺席
- 选修课同样计入总绩点，选评分高的老师
- 重修可以覆盖原成绩（多数学校有此政策，确认本校规定）
- 考前问学长学姐要往年真题和重点，精准备考

## GPA 的用途

- 保研资格（一般要求专业前 15%-30%）
- 奖学金评选
- 出国留学申请（TOP 50 学校一般要求 3.5+）
- 部分企业校招有绩点门槛
