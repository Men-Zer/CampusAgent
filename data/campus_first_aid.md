# 校园急救常识

## 常见急救场景
| 场景 | 处理 |
|------|------|
| 流鼻血 | 头前倾（不要后仰），捏住鼻翼5-10分钟 |
| 烫伤 | 流动冷水冲15-20分钟，不要涂牙膏酱油 |
| 扭伤 | RICE原则：休息+冰敷+加压+抬高 |
| 低血糖晕倒 | 平躺→给糖/巧克力→补充水分 |
| 中暑 | 移到阴凉处→解开衣领→湿毛巾冷敷额头腋下→喝淡盐水 |
| 异物卡喉 | 海姆立克急救法（站在背后，拳头放在肚脐上方，快速向内向上推） |

## 紧急电话
- 急救：120
- 校医院急诊电话（每个学校不同，建议存在手机里）
- 报警：110

## 建议
- 宿舍备一个小药箱：创可贴+碘伏+感冒药+退烧药+肠胃药
- 新生军训期间尤其要注意中暑和扭伤
