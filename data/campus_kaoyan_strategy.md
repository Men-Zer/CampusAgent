# 考研全攻略

## 初试科目（大多数专业）
- 政治（100分）
- 英语（100分）
- 数学/专业课一（150分）
- 专业课二（150分）

## 时间线
| 阶段 | 时间 | 任务 |
|------|------|------|
| 择校择专业 | 大三上 | 查报录比、复试线 |
| 资料准备 | 大三寒假 | 收集参考书、真题 |
| 系统复习 | 大三下-大四上 | 四门课分轮次推进 |
| 网上报名 | 10月 | 研招网yz.chsi.com.cn |
| 现场确认 | 11月 | 各报考点 |
| 初试 | 12月下旬 | 全国统考 |
| 查分 | 次年2月 | 各省教育考试院 |
| 复试 | 次年3-4月 | 目标院校面试 |
