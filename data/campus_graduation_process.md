# 毕业离校全流程

## 大四下学期时间线
| 时间 | 事项 |
|------|------|
| 3月 | 论文定稿，准备查重 |
| 4月 | 查重通过，提交最终版 |
| 5月 | 论文答辩 |
| 6月 | 毕业典礼、领毕业证学位证 |
| 6-7月 | 离校手续（退宿、转档案、转户口） |

## 毕业手续
1. 论文归档（交图书馆）
2. 归还图书、交清欠款
3. 退宿（打扫宿舍、退钥匙）
4. 户籍迁移（户口在学校的选择迁出）
5. 档案转递（到工作单位/人才市场/生源地）
6. 团组织/党组织关系转出

## 报到证（就业报到证）
由学校统一办理，是去单位报到的凭证，务必妥善保管。
