# 出国留学准备

## 时间线
| 时间 | 任务 |
|------|------|
| 大二 | 确定留学方向、开始学托福/雅思 |
| 大二暑假 | 参加夏校/暑研，丰富背景 |
| 大三上 | 考出托福/雅思成绩 |
| 大三下 | 考GRE/GMAT，确定选校名单 |
| 大三暑假 | 找实习/科研，准备文书 |
| 大四上（9-12月） | 提交申请 |
| 大四下（1-4月） | 收offer，决定去向 |
| 5-8月 | 办签证、行前准备 |

## 热门留学目的地
| 国家/地区 | 学制 | 费用/年（参考） |
|-----------|------|----------------|
| 美国 | 1-2年 | 30-50万RMB |
| 英国 | 1年 | 25-40万RMB |
| 澳大利亚 | 1.5-2年 | 20-35万RMB |
| 中国香港 | 1年 | 15-25万RMB |
| 新加坡 | 1-1.5年 | 15-25万RMB |
| 日本 | 2年 | 8-15万RMB |
