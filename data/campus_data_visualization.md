# 数据可视化入门

## 常用工具
| 工具 | 特点 | 免费 |
|------|------|------|
| Excel | 基础图表，人人会用 | ✅ |
| Power BI | 微软出品，交互式仪表盘 | 免费版可用 |
| Tableau Public | 专业可视化 | 免费版可发布 |
| Python(Matplotlib/Plotly) | 代码生成图 | ✅ |
| ECharts | 国内最流行JS图表库 | ✅ |

## 图表示例
- 趋势图→折线图
- 占比→饼图
- 对比→柱状图
- 分布→直方图/箱线图
- 关系→散点图

## 学习资源
- ECharts官方示例：echarts.apache.org/examples
- Datawrapper（datawrapper.de）：零代码做图
- Flourish（flourish.studio）：动态数据可视化
